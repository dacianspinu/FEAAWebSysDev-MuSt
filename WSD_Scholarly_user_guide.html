1. Step-by-Step User Guide

  1. User registers for the service through the registration form. They will provide a valid email address, a password of at least 8 characters and its confirmation. After the correct completion of the inputs in the form they will click the “REGISTER” button located underneath the form.
  2. User will receive a confirmation email that will contain a redirect link to the application. When the application detects that link it will get the encoded data from the link and will send it to server.
  3. User is redirected to the login account. He will input the used email address and password and will be able to access the application features.
  4. User is redirected to the preferences setup. Here he will see all the categories that we have and he can choose from them to act as base for the the playlist generation. He must select at least 5 categories. By using the search box he can filter the artists, albums and genres that appear on the screen. When the user clicks the “Done” button the user will be redirected to the streaming platform login.
  5. On the streaming platform login user sees buttons with each platform. On clicking the button a platform-specific login page appears and from there the user logs in. After login the user is redirected to the playlists page. If he doesn’t want to login in any streaming platform when clicking the “Skip” button he will be redirected to the playlists page
  6. From the home view the user can access a playlist’s details, with its tracks and can click on a track to view more details about it and also we will prompt new artists and genres based on the playlist details. Clicking outside this modal the app will automatically lead to its closing. Also if clicking the Escape key will do the same.
  7. From the home view the user can access his preferences by clicking on the icon on the upper right corner. From here he can edit his preferences and the playlists that are seen in the home view are also edited accordingly.
